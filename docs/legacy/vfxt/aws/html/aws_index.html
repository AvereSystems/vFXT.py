Avere vFXT installation guide for Amazon Web Services¶

This document set describes how to create and manage an Avere OS vFXT cluster on the Amazon Web Services (AWS) cloud computing service.

Additional documentation¶

Additional information about configuring and operating an Avere cluster is provided in the Cluster Configuration Guide. Visit http://azure.github.io/Avere/ for release notes and other information.
